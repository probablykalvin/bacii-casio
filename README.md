# BACCASIO

A fast and accurate BACII grade calculator. 

👉 **[Try it out here!](https://probablykalvin.github.io/bacii-casio)**

## Features

- **Double-Track Support**: Calculates scores for both the Science and Social Science tracks.
- **Detailed Breakdown**: Displays exact percentages and the pass/fail status for each subject.
- **Save & Share**: Save an unlimited number of score profiles locally, share your scores via generated short links, and download high-resolution report cards.
